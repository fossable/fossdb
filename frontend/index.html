REST API Reference

Complete reference for the FossDB API endpoints

Base URL

https://api.fossdb.org/v1
GET /packages

List all packages with optional filtering and pagination

Query Parameters: page, limit, search, tag
GET /packages/{id}

Get detailed information about a specific package

POST /packages

Create a new package entry (requires authentication)

Authorization: Bearer token required
POST /auth/register

Register a new user account

POST /auth/login

Authenticate with email and password
