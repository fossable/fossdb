REST API

JSON API for querying package metadata

Endpoints

Base URL: http://localhost:3000/api
GET /stats

Get real-time database statistics

{
  "total_packages": 42,
  "total_versions": 128,
  "total_users": 5,
  "total_vulnerabilities": 0,
  "total_timeline_events": 15,
  "scrapers_running": ["crates.io"]
}
GET /packages

List packages (with optional search/filtering)

Query Parameters: search, page, limit, tag
GET /packages/{id}

Get package details by ID

POST /packages AUTH

Create a new package entry

Headers:
Authorization: Bearer YOUR_JWT_TOKEN
POST /auth/register

Register new user (returns JWT token)

Body: username, email, password
POST /auth/login

Authenticate (returns JWT token)

Body: email, password
GET /analytics

Calculated analytics (language/license distribution, security stats)

Notes

  • • All responses are JSON
  • • JWT tokens expire after 7 days
  • • Protected endpoints return 401 if token is missing/invalid
  • • Configure base URL via FOSSDB_API_URL environment variable
